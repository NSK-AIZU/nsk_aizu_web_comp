# 株式トレード支援システム

## 概要

トレンド銘柄を見つけて、適切なタイミングで売却するための支援システム。

## 役割分担

| 判断 | 担当 | 備考 |
|------|------|------|
| **銘柄を探す** | コード | `scan_trend.py` でトレンド銘柄をスクリーニング |
| **買う銘柄を決める** | あなた | スクリーニング結果とテーマを見て判断 |
| **いつ買うか** | あなた | トレンド銘柄は見つけたら即買い |
| **いつ売るか** | コード | トレーリングストップで判断 |

## 投資の流れ

```
┌─────────────────────────────────────────────────────────────┐
│ Step 1: トレンド銘柄を探す                                  │
│         python scan_trend.py --index defense                │
│         python scan_trend.py --index ai                     │
│         python scan_trend.py --index nuclear                │
├─────────────────────────────────────────────────────────────┤
│ Step 2: リストを見て買う銘柄を決める                        │
│         → スコア75点以上の「強いトレンド」が候補            │
│         → 自分の投資テーマに合うか確認                      │
├─────────────────────────────────────────────────────────────┤
│ Step 3: 決めたらすぐ買う                                    │
│         → トレンド銘柄は押し目を待つと機会損失              │
├─────────────────────────────────────────────────────────────┤
│ Step 4: 保有中は定期的にトレンド確認                        │
│         python find_trend.py                                │
│         → 「強いトレンド」を維持しているか確認              │
├─────────────────────────────────────────────────────────────┤
│ Step 5: トレンドが弱まったら売却を検討                      │
│         → スコアが50点以下になったら売却候補                │
│         → トレーリングストップに引っかかったら売却          │
└─────────────────────────────────────────────────────────────┘
```

## トレンド判定基準

| スコア | 判定 | アクション |
|--------|------|------------|
| 75点以上 | 🚀 強いトレンド | 買い候補 / 保有継続 |
| 50-74点 | 📈 弱いトレンド | 様子見 |
| 25-49点 | ➡️ 横ばい | 売却検討 |
| 25点未満 | 📉 下降トレンド | 売却 |

## 投資テーマ

現在の投資テーマと対応するセクター：

| テーマ | 理由 | スキャンコマンド |
|--------|------|------------------|
| 金融 | 利上げ継続 | `--index high_dividend` |
| 防衛 | 防衛費増額 | `--index defense` |
| AI | AI需要爆発 | `--index ai` |
| 原発 | 原発再稼働 | `--index nuclear` |
| 半導体 | AI/データセンター需要 | `--index semiconductor_ex` |

## ツール一覧

### 1. トレンド銘柄スキャナー（scan_trend.py）

新しいトレンド銘柄を探す。

```bash
# セクター別スキャン
python scan_trend.py --index defense          # 防衛関連
python scan_trend.py --index ai               # AI関連
python scan_trend.py --index nuclear          # 原発関連
python scan_trend.py --index semiconductor_ex # 半導体関連
python scan_trend.py --index nikkei225        # 日経225全体

# 利用可能なセクター一覧
python scan_trend.py --list

# 結果をCSV保存
python scan_trend.py --index defense --save
```

### 2. トレンド判定（find_trend.py）

保有銘柄や特定銘柄のトレンドを確認。

```bash
# 保有銘柄全体をチェック
python find_trend.py

# 特定銘柄をチェック
python find_trend.py 7011.T 7013.T
```

### 3. データダウンロード（download_data.py）

バックテスト用のデータをダウンロード。

```bash
python download_data.py
```

## バックテストから得た教訓

1. **銘柄選びが最重要** - トレンド銘柄 vs 横ばい銘柄で21倍の差
2. **トレンド銘柄は即買い** - 押し目を待つと機会損失（-150%〜-350%）
3. **横ばい銘柄は避ける** - 手間の割にリターンが低い
4. **売りはコードに任せる** - トレーリングストップで機械的に判断
