# NSK_AIZU ホームページ改善計画書

## はじめに ― なぜ改善が必要か

現在のホームページは技術的に丁寧に作られている（レスポンシブ対応、多言語、microCMS連携、構造化データ）。しかし、以下の2つの視点で課題がある。

**視点① 親目線で「応援したくなる」か？**

現代のファン心理は「完成品を消費する」から「成長過程を一緒に見守る」へ変化した。アイドルの世界でも、完成された姿よりも駆け上がっていく過程に人は惹かれる。NSK_AIZUのサイトには、この「成長の過程」「人の気配」「泥臭い日常」がほぼ見えない。写真は3枚のみ（logo.png, ogp.jpg, vision.jpg）。メンバーの顔も、畑で汗を流す姿も、試行錯誤の記録もない。

**視点② 抽象的に、ふわっと簡潔にわかりやすいか？**

衆議院選挙の結果が示したように、人は情報量ではなく「感覚的にわかる短い言葉」で動く。トップページのコピー「会津の未来を、耕す。」は秀逸。しかし下層ページ（VISION, ABOUT）に入ると、統計データ（1万2千人、18.33%、3,000㎡）と長文の説明が並び、感情的な共感より「報告書」の印象が強い。

|---|---|---|

## 改善の全体方針

1. **黒ベースのデザインは維持**しつつ、ウォームゴールド（#D4AF37）のアクセントカラーを追加して温かみを出す
2. **文章量を半分以下に削減**し、短い言葉で感情に訴える構成へ
3. **人・過程・日常**を見せるコンテンツをサイトの前面に出す
4. **ギャラリーと活動記録**をトップページから直接アクセスできるようにする

## Phase 1: デザイン基盤の更新

**対象ファイル:** `style.css`

### やること

CSS変数でウォームゴールドのアクセントカラーを定義し、全ページに適用する。

```css
:root {
    --color-gold: #D4AF37;
    --color-gold-light: #E5D4AA;
    --color-accent-subtle: rgba(212, 175, 55, 0.05);
}
```

### 具体的な変更点

- カードのホバー時の枠線色を `#666` → ゴールドに変更
- ボタンのホバー時にゴールドの微細なシャドウを追加
- 行間を広げ（`line-height: 2.0`以上）、セクション間のpaddingを増やして「ふわっとした」呼吸感を演出
- 見出しサイズを `clamp()` で段階的に調整

### 期待される効果

冷たい印象の黒背景に温かみが加わり、「応援したくなる」雰囲気の土台ができる。

## Phase 2: VISIONページの簡素化

**対象ファイル:** `vision.html`, `style.css`

### 削除する内容

- 「この10年間で約1万2千人の人口が減少」
- 「18.33％に達する市内の空き家率」
- 「全国平均を大きく上回る」
- その他の統計データ・数値表現

### 置き換えの方向性

統計ではなく、風景の描写と想いで語る。例：

> 減りゆく街で、新しい価値を育む。
> かつて誰かの家族がいた空き家。手入れされなくなった田畑。
> その風景を、もう一度輝かせたい。

### CSSの調整

- `.vision-text` の font-size を 16px → 18px、line-height を 2.2 → 2.4 に
- max-width を 750px に絞り、読みやすさを向上
- 段落間に装飾的な区切り（細いゴールドライン等）を入れて呼吸感を演出

## Phase 3: ABOUTページの書き直し

**対象ファイル:** `about.html`, `style.css`

### 現状の問題

MISSION / STRENGTH / FOR WHOM の各セクションが2段落ずつの長文で、「再現性のある栽培法」「総面積約3,000㎡」などの専門的・数値的な表現が多い。

### 改善方針

**MISSIONセクション:** 2段落の説明文 → 短い詩的表現へ圧縮

```
負の遺産を、未来の資源へ。
減りゆく街で、価値を耕す。
```

**STRENGTHセクション:** 長文 → 3枚のビジュアルカードへ

- 「データサイエンス」「フィジカル」「若い力」の3カード
- 各カード：見出し1行 + 説明1文のみ
- 左ボーダーにゴールドのアクセント

**FOR WHOMセクション:** 長文 → シンプルなメッセージへ

```
後継者不足に悩む農家さんへ。
そして何より、未来の担い手たちへ。
「農業に興味はあるけど、関わり方がわからない」
その最初の窓口になりたい。
```

**新規追加: Our Journey（タイムライン）セクション**

- NPO設立から現在までのマイルストーンを時系列で表示
- 左ボーダーライン + 日付 + 短い説明
- 「成長の過程」を視覚的に見せる ← 親目線への直接的な回答

## Phase 4: トップページの再構成

**対象ファイル:** `index.html`, `style.css`, `common.js`

### 現状

4カードのグリッドレイアウト（VISION / NEWS / ABOUT / CONTACT）。100dvhでスクロール不可。

### 改善案

100dvhのロック解除し、縦スクロール型の構成に変更する。

| セクション | 内容 | 目的 |
|---|---|---|
| 1. ヒーロー | 「会津の未来を、耕す。」+ 活動写真の背景 | 第一印象。感情的なフック |
| 2. ギャラリープレビュー | 日めくりギャラリーから最新3枚を表示 | 人の気配・日常を「最初に」見せる |
| 3. 活動タイムライン | 最近の取り組みを時系列で3件 | 成長過程の可視化 |
| 4. ストーリー付き数字 | 農地面積 / メンバー数 / 次の目標（3カード） | 統計をエモーショナルに伝える |
| 5. NEWS | 最新2件（現在と同様） | 更新感の維持 |
| 6. ナビカード | VISION / ABOUT / CONTACT への導線 | 回遊性の確保 |

### 技術的なポイント

- `.home` の `height: 100dvh; overflow: hidden;` を解除
- ギャラリープレビューは `gallery/data.json` からJS読み込み
- 活動タイムラインは静的HTML、またはmicroCMSの新規エンドポイントから取得

- 活動日誌ページへのリンクを追加（Phase 6で作成）

## Phase 6: 活動日誌ページの新規作成

**新規ファイル:** `activity.html`, `activity.js`
**追加スタイル:** `style.css` に新コンポーネント追加

### 目的

「親目線で応援したくなる」の核となるコンテンツ。日々の活動・奮闘を写真と短文で公開する場所。

### ページ構成

- ヒーロー：「- ACTIVITY DIARY -」「学生たちの日々の記録」
- カテゴリフィルター：すべて / 田畑 / チーム / イベント
- カードグリッド：写真 + 日付 + タイトル + カテゴリタグ

### 技術実装

- microCMSに「activity」エンドポイントを新規作成
  - フィールド：title, date, image, category, description
- `activity.js` で fetch → DOM生成
- カテゴリフィルターはdata属性ベースのJSフィルタリング

### デザイン

- カードにゴールドのカテゴリタグ
- ホバー時にボーダーがゴールドに変化
- レスポンシブ：PC 3列 → タブレット 2列 → スマホ 1列
