# Webサイト開発仕様書（改訂版 Ver 1.1）

プロジェクト名： 祖母の絵と母の切手 日めくりギャラリー（仮）

参照UI: 提示されたダークモードのWebサイト画像 (image_0.png)

## 1. プロジェクトの概要

（変更なし）

祖母が描いた絵画と、母が収集した切手コレクションをデジタルアーカイブ化し、Web上で公開する。

「1日に1作品だけ、ランダムに」表示することで、毎日訪れる楽しみと、一つ一つの作品と向き合う時間を提供する。

## 2. ターゲット・目的

（変更なし）

- **ターゲット:** 親族、友人、および偶然訪れたWeb閲覧者。
- **目的:** 家族の記憶の保存、作品の再評価、静かな鑑賞体験の提供。

## 3. コンテンツ要件（AI連携）

（変更なし）

画像に加え、生成AI（Claude等）によって作成されたテキストを併記する。

| **カテゴリ** | **画像** | **テキストの内容（生成AIによる作成方針）** |
| --- | --- | --- |
| **祖母の絵** | スキャン/撮影画像 | **「想い・ストーリー」**（詩的で温かい150文字程度のストーリー） |
| **母の切手** | スキャン画像 | **「時代背景・トリビア」**（発行年や事象に関する簡潔な解説） |

## 4. 機能要件

### 4.1. フロントエンド（表示機能）

- **日次更新ロジック（変更なし）:**
    - アクセスした「年月日」をキーとして、表示する作品を決定する（同日は同じ作品）。
- **表示要素（デザインに合わせて整理）:**
    - **ヘッダー:** サイトタイトル（左上）。（※言語切替やメニューは初期段階では不要とするが、デザイン上配置しても良い）
    - **メインエリア:**
        1. **作品画像:** 画面中央に大きく配置。
        2. **テキストオーバーレイ:** 画像の上に重なる半透明のボックス。
            - 作品タイトル
            - 解説テキスト

### 4.2. データ管理

- **データ形式:** `data.json` (単一JSONファイル) で管理。（変更なし）

## 5. デザイン・UI要件【大幅変更箇所】

ご提示の画像をリファレンスとし、没入感のあるダークテーマを採用する。

- **コンセプト:** 「没入感」「静謐」「美術館の暗室」「モダン＆クラシック」
- **配色:**
    - **背景色:** 深い黒（例: `#0a0a0a` または `#111111`）。
    - **文字色:** オフホワイト（例: `#F0F0F0`）。真っ白は避け、目に優しく上品な色合いにする。
    - **テキストボックス背景:** 半透明の黒（例: `rgba(0, 0, 0, 0.6)`）。画像がうっすら透けて見えるようにする。
- **フォント:**
    - **日本語:** 上品な明朝体・セリフ体を採用する（Google Fonts `Shippori Mincho` や `Noto Serif JP` 等を想定）。
    - テキストは基本的に**中央揃え**とする。
- **レイアウト構造:**
    - **ヘッダー:** シンプルにサイトロゴ/タイトルのみを左上に配置する。（参考画像のようにスッキリとさせる）
    - **メインビジュアル:** 作品画像を画面の大部分を使って大きく表示する。
    - **テキストオーバーレイ:**
        - 画像の右下（または中央付近）に、テキスト用のボックスを重ねて配置する（position: absolute等を使用）。
        - ボックス内にはタイトルと解説文を収め、適切な余白を持たせる。
